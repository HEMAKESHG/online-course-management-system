
# 📘 Online Course Management System (Console-Based Java)

## 🚀 Project Overview

The **Online Course Management System** is a console-based Java application designed to simulate an educational platform with three user roles:

* 👨‍💼 Admin
* 👨‍🏫 Instructor
* 👨‍🎓 Student

The system demonstrates real-world entity relationships and core Object-Oriented Programming (OOP) concepts.

---

## 🎯 Features

### 👨‍💼 Admin

* 🔐 Password-protected login
* ➕ Add instructors
* 👀 View all registered students
* 📚 View all available courses

### 👨‍🏫 Instructor

* 🔑 Login using Instructor ID
* 📘 Create new courses
* 📋 View created courses

### 👨‍🎓 Student

* 📝 Register with ID and name
* 🔑 Login using Student ID
* 📚 View available courses
* ✅ Enroll in courses

---

## 🏗️ System Design Concepts Used

* Object-Oriented Programming (OOP)

  * Encapsulation
  * Composition
  * Role-based control
* Java Collections (`ArrayList`)
* Console-based interaction (`Scanner`)
* Basic authentication logic
* Real-world relationship mapping:

  * Instructor → Course (1:N)
  * Student ↔ Course (M:N)
